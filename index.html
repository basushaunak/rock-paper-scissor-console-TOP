Rock Paper Scissor (Console Based) Game for The Odin Project

declare an array of strings with rock, paper and scissors as three elements.

declare a variable named userinput of type string

declare a variable called userChoice as an integer

declare a variable called computerChoice as an integer

Ask the user to enter rock, paper or scissors and store it in userinput variable

change the userinput to lowercase

if user enter rock, store 0 in userChoice,

if user enters paper store 1 in userChoice,

if user enters scissors store 2 in userChoice.

if user enters something else, ask the user to enter either rock, paper or scissors

ask the computer to randomly return 0(rock), 1 (paper) or 2 (scissor).

if both user and computer selects the same thing, it is a tie

if user enters rock but the computer selects paper the computer wins

if the user enters paper and the computer selects scissor the computer wins<

in all other cases the player wins

(user selects paper, the computer selects rock & the user selects scissors and the computer selects paper)
